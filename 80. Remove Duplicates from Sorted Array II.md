<h3>题意<h3>
<p>给一个sorted list,如果里面重复的元素大于2，那么就把第三到第N个元素删除。
return 新的list的长度<p>

<h3>解题<h3>
<p>            if nums[i] == nums[i-2]:
                del nums[i]
<p>

```python
class Solution(object):
    def removeDuplicates(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        
        if len(nums) <= 2:
            return len(nums)
         
        i = 2
        while i < len(nums):
            if nums[i] == nums[i-2]:
                del nums[i]
            else:
                i += 1
        return len(nums)
```
